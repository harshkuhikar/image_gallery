🚀 QUICK START (3 STEPS):

Step 1: Upload Project (5 min)
   → Compress project folder to ZIP
   → Upload to Hostinger File Manager
   → Extract to /var/www/sonal-agritech

Step 2: Run Scripts (5 min)
   → Open SSH Terminal in Hostinger
   → cd /var/www/sonal-agritech
   → chmod +x *.sh
   → ./setup-server.sh (first time only)
   → ./auto-deploy.sh

Step 3: Configure DNS & SSL (5 min)
   → Add A record in Hostinger DNS
   → Wait 5-30 minutes
   → Run: ./quick-ssl.sh
   → Done! 🎉
