═══════════════════════════════════════════════════════════════
  QUICK DEPLOYMENT STEPS
═══════════════════════════════════════════════════════════════

1. Connect to VPS: ssh root@your-vps-ip
2. Install Node.js, PM2, Nginx
3. Upload project to /var/www/sonal-agritech
4. Create .env file with credentials
5. Run: npm install
6. Run: cd client && npm install --legacy-peer-deps && npm run build
7. Start: pm2 start server/index.js --name sonal-agritech
8. Configure Nginx for linktree.sonalagritech.co.in
9. Setup SSL: sudo certbot --nginx -d linktree.sonalagritech.co.in
10. Configure DNS in Hostinger (A record: linktree → VPS IP)

Time Required: ~30 minutes
